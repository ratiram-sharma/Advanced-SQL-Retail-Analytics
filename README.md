# Advanced SQL E-Commerce & Retail Analytics Pipeline

An enterprise-grade SQL portfolio demonstrating advanced data analytics capabilities, including cohort analysis, customer segmentation, window functions, and database performance tuning.

## Business Case Studies Solved:
* **Case Study 1 (Cohort Analysis):** Developed a window-function-driven pipeline tracking baseline user behavior and registration milestones.
* **Case Study 2 (RFM Segmentation):** Built a multi-layered Common Table Expression (CTE) classifying customer lifetime value into actionable tiers (VIP, Regular, At-Risk).
* **Case Study 3 (Product Performance):** Engineered nested ranking algorithms using `DENSE_RANK()` to isolate top 3 revenue-generating products per catalog category.
* **Case Study 4 (Performance Optimization):** Optimized inventory metrics scanning 100M+ records using composite indexing structures and operational decoupled database views.

## Tech Stack Used:
* Advanced SQL (PostgreSQL / MySQL compliant)
* Window Functions (`MIN() OVER`, `DENSE_RANK()`)
* Common Table Expressions (Nested CTEs)
* Indexing & Database Views
